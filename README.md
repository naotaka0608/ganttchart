# Gunshart - ガントチャートアプリケーション

Python + PySide6 + SQLiteで作成したガントチャート管理アプリケーション

## 機能

- 多階層タスク管理（無制限のネスト）
- タスクの展開/折りたたみ
- 進捗率の表示
- マイルストーン対応
- タスク間の依存関係
- 視覚的なガントチャート表示
- SQLiteによるデータ永続化

## プロジェクト構造

```
gunshart/
├── main.py              # アプリケーションエントリーポイント
├── requirements.txt     # 依存パッケージ
├── database/
│   ├── __init__.py
│   ├── db_manager.py    # SQLite操作
│   └── schema.sql       # テーブル定義
├── models/
│   ├── __init__.py
│   ├── task.py          # タスクモデル
│   └── project.py       # プロジェクトモデル
└── views/
    ├── __init__.py
    ├── main_window.py   # メインウィンドウ
    ├── task_tree.py     # サイドバーのツリービュー
    └── gantt_chart.py   # ガントチャート描画
```

## セットアップ

### 1. 仮想環境の作成（uv使用）

```bash
uv venv
```

### 2. 依存パッケージのインストール

```bash
uv pip install PySide6
```

### 3. アプリケーションの起動

```bash
# Windows
.venv/Scripts/python.exe main.py

# Linux/Mac
.venv/bin/python main.py
```

## 使い方

### タスクの追加

1. ツールバーの「新規タスク」ボタンをクリック
2. タスク名、開始日、終了日、進捗率などを入力
3. OKボタンで保存

### タスクの編集

- タスクツリーで右クリック → 「編集」

### 子タスクの追加

- 親タスクを右クリック → 「子タスク追加」

### タスクの削除

- タスクを右クリック → 「削除」
- 子タスクも一緒に削除されます

### 展開/折りたたみ

- ツリービューで▶アイコンをクリック

### マイルストーン

- タスク編集ダイアログで「マイルストーン」にチェック
- ガントチャートで赤色で表示されます

## データベース

- アプリケーション起動時に `gunshart.db` が作成されます
- SQLiteで管理されているため、バックアップも簡単です
